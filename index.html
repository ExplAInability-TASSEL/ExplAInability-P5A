Welcome to ExplAInability map !

This map helps getting insights on how the TASSEL algorithm works.

The two buttons you can see above enable to hide and show the labeled segments and the heatmap.
The heatmap represents the attention weights of the algorithm. The more the weight is close to 1, the more the algorithm is sure of the class of the segment.

When clicking on a segment, you can see the class of the segment and the attention weights of the algorithm for each pixel of the segment in a right side panel.

The legend you can see on the left side of the map represents the different classes of the segments.

If you have any question, please contact us using the contact button at the top.
